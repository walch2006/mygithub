问：如何用git上传代码到github？

1.在本地安装git，并配置好本地的git信息； （要配置本地的Git信息，首先需要安装Git，安装完成后，执行以下命令： git config --global user.name “[Your Name]” git config --global user.email “[Your Email]” 替换[Your Name]和[Your Email]为您的名字和邮箱。 这将设置您的全局Git配置，以便您在每次提交代码时都会有一个明确的标识。）

2.在github上创建一个新的仓库，将仓库的地址复制下来； （要复制GitHub仓库地址，可以点击仓库的“Clone or Download”按钮，然后复制“Clone with HTTPS”的地址。）

3.在本地建立一个文件夹，并将需要上传的文件放入文件夹；

4.进入该文件夹，使用git bash，执行git init，将该文件夹变成git仓库；

5.执行git add .，将文件加入暂存区；

6.执行git commit -m '提交信息'，将文件提交到本地仓库；

注：执行git commit时，进入vim，此时要先按a或i，才能进入编辑，输入说明如第一次，然后要先按Eｓｃ退出编辑，再按英文的“：”，输入“ｗｑ”，回车一ｇｉｇ

7.执行git remote add origin 仓库地址，将本地仓库与github仓库关联；

8.执行git push -u origin master，将代码提交到github仓库；

参考git、github 保姆级教程入门，工作和协作必备技术，github提交pr - pull request
